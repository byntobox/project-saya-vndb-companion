# Branding Asset Drop Folder

Replace the placeholder files in this folder with your real logo/icon assets.

Required files used by the app:

- `favicon.svg` (browser tab icon, SVG)
- `favicon-32x32.png` (browser tab fallback icon)
- `apple-touch-icon.png` (iOS home screen icon)
- `pwa-192x192.png` (PWA icon)
- `pwa-512x512.png` (PWA icon / splash usage)

Recommended source workflow:

1. Start from a square master logo (at least 1024x1024).
2. Export PNGs for the sizes above.
3. Export SVG for `favicon.svg` if your mark is vector-safe.

Current files are placeholders so references resolve during development.
